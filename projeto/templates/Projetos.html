Quem sou eu Interesses Projetos
Meu primeiro projeto está sendo desenvolvido na FATEC de São José dos Campos/SP. Onde participo de um grupo, responsável por desenvolver um Sistema de Divulgação de Informações Prioritárias da FATEC. Esse projeto é tratado como API - Aprendizado por Projeto Integrado.
Neste sistema, está sendo desenvolvido as seguintes páginas web:
  1. Tela de login
  2. Tela de cadastro
  3. Tela inicial (onde estará disponibilizando prévias de informações)
  4. Tela cheia (agrupamento de informações)
  5. Tela de visualização da informações
